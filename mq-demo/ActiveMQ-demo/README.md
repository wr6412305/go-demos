# README

启动ActiveMQ前需要先装java

```sh
wget http://apache.fayea.com/activemq/5.15.10/apache-activemq-5.15.10-bin.tar.gz
tar zxf apache-activemq-5.15.10-bin.tar.gz
mv apache-activemq-5.15.10 activemq-5.15.10
cd activemq-5.15.10/bin
./activemq start
```

```sh
./activemq stop
./activemq status
```
